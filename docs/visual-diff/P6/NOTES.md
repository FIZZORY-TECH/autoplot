```bash
node scripts/capture-visual-diff-p6.mjs
```

Prerequisites:
- `npm run dev` (port 1420) should be running, or the script starts it.
- `@playwright/test` must be installed (`npm install`).

## Prototype comparison

`panel-research-with-dataset.prototype.png` is generated from
`app-design/project/autoplot.html` when it is present in the repo.
All other captures are rebuild-only — the prototype's internal state (tool
round-trips, dataset cards, library tabs) cannot be driven headlessly from
the outside without the `__aiCapture` escape hatch installed by
`src/ai/__capture_helpers.ts`.
