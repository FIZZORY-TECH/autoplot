# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it **privately** by
emailing **security@fizzory.com**. Do not open a public GitHub issue for
security reports.

As an alternative, you may use GitHub's private vulnerability reporting via the
repository's **Security** tab.

## Scope and Expectations

autoplot is a **beta** research and paper-trading tool. It is research-only and
never places real orders.

- There is **no bug-bounty program** and no monetary reward for reports.
- Responses are **best-effort only**; there is no guaranteed response time.

We appreciate responsible disclosure and will do our best to acknowledge and
address valid reports.
